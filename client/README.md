# AceAcademics Frontend
## Professional Academic Productivity Platform for Competitive Programming

AceAcademics is a desktop-first productivity platform designed for students balancing competitive programming with college coursework.

### Features
- **Dashboard**: Data-first stats overview with LeetCode integration.
- **Weekly Planner**: Manage CP and academic goals with progress tracking.
- **AI Mentor**: Personalized insights powered by Gemini.
- **Settings**: Profile management and LeetCode sync preferences.

### Tech Stack
- React 18 + Vite + TypeScript
- Tailwind CSS (Utility-first styling)
- Recharts (Data visualization)
- Lucide React (Icon library)
- Motion (Animations)
- @google/genai (AI logic)

### Installation
1. `npm install`
2. `npm run dev`

### Environment Variables
Requires a `GEMINI_API_KEY` in your environment or `.env` file to power the AI Mentor features.
